# Gyank

A lightweight clipboard manager for Linux, built with Go.

Gyank monitors clipboard changes, keeps a local history of copied content, and allows users to quickly access and reuse previous clipboard items.

## Features

- 📋 Clipboard history tracking
- ⚡ Lightweight background process
- 🔍 Quick access to previously copied text
- 🖥️ Native desktop interface
- 💾 Local storage
- 🐧 Built for Linux

## How it works

Gyank runs in the background and watches for clipboard changes. Whenever new content is copied, it is captured and stored in the clipboard history.

Users can open the interface to browse previous clipboard entries and copy them again when needed.
